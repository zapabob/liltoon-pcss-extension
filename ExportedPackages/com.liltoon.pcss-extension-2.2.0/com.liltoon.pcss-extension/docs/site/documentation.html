📖 lilToon PCSS Extension ドキュメント

VRChat用高品質PCSS (Percentage-Closer Soft Shadows) ソリューションの完全ガイド

🎯 概要

lilToon PCSS Extension は、VRChat向けに特別に最適化された高品質なソフトシャドウシステムです。 lilToon v1.10.3との完全互換性を保ちながら、PCSS (Percentage-Closer Soft Shadows) 技術を 統合し、リアルな影表現を実現します。

ℹ️ 重要な特徴
  • 完全統合: lilToon との seamless な統合
  • VCC対応: VRChat Creator Companion でワンクリックインストール
  • Quest最適化: 自動パフォーマンス調整機能
  • ModularAvatar対応: 非破壊セットアップ

✨ 機能一覧

🎨 シェーダー機能

機能 説明 対応プラットフォーム
PCSS Shadow 高品質なソフトシャドウレンダリング PC / Quest
Dynamic Quality デバイス性能に応じた自動品質調整 PC / Quest
Real-time Control VRChat内でのリアルタイム調整 PC / Quest
VRC Light Volumes 次世代ボクセルライティング統合 PC / Quest

🔧 開発者機能

  • VCC Setup Wizard: 自動セットアップウィザード
  • Performance Profiler: リアルタイムパフォーマンス監視
  • Expression Menu Creator: 自動エクスプレッションメニュー生成
  • Compatibility Manager: lilToon バージョン互換性管理

📦 インストール

🚀 VCC経由でのインストール（推奨）

✅ 最も簡単な方法
  1. VRChat Creator Companion を開く
  2. Settings → Packages → Add Repository をクリック
  3. 以下のURLを追加:
https://zapabob.github.io/liltoon-pcss-extension/index.json

🚀 VCCに自動追加

💾 手動インストール

  1. 最新版をダウンロード: ダウンロード
  2. UnityPackage を Unity プロジェクトにインポート
  3. VCC Setup Wizard が自動で起動
  4. セットアップを完了

⚙️ セットアップ

🎮 基本セットアップ

  1. Setup Wizard 起動 
    Unity Editor → Window → lilToon PCSS Extension → Setup Wizard
  2. lilToon 互換性チェック

    自動でlilToonのバージョンを検出し、互換性を確認します。

  3. VRChat SDK 統合

    VRChat SDK3 との統合設定を自動で行います。

  4. パフォーマンス設定

    プロジェクトに最適なパフォーマンス設定を適用します。

⚠️ 注意事項
  • lilToon v1.10.3 以降が必要です
  • VRChat SDK3 (Avatars) が必要です
  • Unity 2019.4.31f1 以降を推奨します

🎯 使用方法

🎨 マテリアル設定

  1. lilToon マテリアル作成 
    Project → Create → Material
Inspector → Shader → lilToon/PCSS Extension
  2. PCSS 設定
    • Shadow Quality: 影の品質設定
    • Shadow Distance: 影の描画距離
    • Shadow Bias: 影のバイアス調整
  3. VRChat 最適化

    Quest対応の自動最適化が有効になります。

🎮 VRChat内での制御

エクスプレッションメニューから以下を制御できます:

  • Shadow Intensity: 影の強度 (0-100%)
  • Shadow Quality: 影の品質 (Low/Medium/High)
  • Dynamic Adjustment: 動的品質調整のON/OFF

⚡ パフォーマンス最適化

🎯 自動最適化機能

  • Platform Detection: PC/Quest自動判定
  • LOD System: 距離に応じた品質調整
  • Frame Rate Monitoring: フレームレート監視による動的調整

🆘 トラブルシューティング

🔧 よくある問題

❌ シェーダーコンパイルエラー

🛠️ 解決方法
  1. Unity Editor を再起動
  2. Assets → Reimport All を実行
  3. Setup Wizard を再実行

❌ VRChatで影が表示されない

  1. VRChat Settings → Graphics → Shadows が有効か確認
  2. アバターのパフォーマンスランクを確認
  3. エクスプレッションメニューでShadow Intensityを調整

❌ パフォーマンス問題

  1. Shadow Quality を下げる
  2. Shadow Distance を短くする
  3. Dynamic Adjustment を有効にする

🔗 API リファレンス

📝 主要クラス

PCSSUtilities

// 影の品質を動的に変更
PCSSUtilities.SetShadowQuality(ShadowQuality.Medium);

// パフォーマンス監視開始
PCSSUtilities.StartPerformanceMonitoring();

// 最適化設定適用
PCSSUtilities.ApplyOptimizationSettings();

VRChatPerformanceOptimizer

// Quest最適化の有効化
VRChatPerformanceOptimizer.EnableQuestOptimization();

// フレームレート監視
VRChatPerformanceOptimizer.MonitorFrameRate(targetFPS: 90);

🎮 エクスプレッションパラメーター

パラメーター名 型 範囲 説明
PCSS_Intensity Float 0.0 - 1.0 影の強度
PCSS_Quality Int 0 - 2 影の品質 (0:Low, 1:Medium, 2:High)
PCSS_Dynamic Bool true/false 動的調整の有効/無効
