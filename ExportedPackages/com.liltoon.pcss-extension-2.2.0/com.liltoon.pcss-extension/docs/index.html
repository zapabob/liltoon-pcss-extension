üöÄ Quick Start

  1. Add this repository to your VCC
  2. Install the package in your VRChat project
  3. Select your avatar's root GameObject
  4. Go to Tools/lilToon PCSS Extension/Advanced Integration/Setup Advanced Integration
  5. All systems will be automatically set up!
