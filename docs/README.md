## 🚀 クイックスタート

### VCCでのインストール

1. **VCCを起動**
2. **Settings** → **Packages** → **Add Repository**
3. **以下のURLを入力**:
   ```
   https://zapabob.github.io/liltoon-pcss-extension/index.json
   ```
4. **Add**をクリック
5. プロジェクトで**lilToon PCSS Extension**を検索してインストール
