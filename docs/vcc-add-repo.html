📋 VCC手動追加手順

  1. VRChat Creator Companion (VCC) を開く
  2. Settings → Packages をクリック
  3. Add Repository ボタンをクリック
  4. 以下のURLをコピーして貼り付け
  5. Add ボタンをクリック
https://zapabob.github.io/lilToon-PCSS-Extension/index.json

🎯 パッケージインストール

  1. VCCでプロジェクトを開く
  2. Manage Project をクリック
  3. Add Package セクションを確認
  4. "lilToon PCSS Extension - All-in-One Edition" を選択
  5. Install ボタンをクリック
📖 詳細インストールガイド 🏠 メインサイトに戻る
⚡ VCCで直接開く

🔧 トラブルシューティング

  • VCCが開かない場合: VCCが正しくインストールされているか確認
  • リポジトリが見つからない場合: URLが正確にコピーされているか確認
  • ダウンロードエラー: インターネット接続を確認し、数分後に再試行
  • その他の問題: サポートページをご確認ください
